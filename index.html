📤 Publish Your Game on Raking Productions

Follow this step-by-step guide to export your game, upload it to itch.io, and get it listed on our platform — completely free.

Select your game engine to see specific export instructions:

🛠️ Phase 1 — Export Your Godot Game for Web
1

Install the Web Export Template

In Godot 4, go to Editor → Export → Manage Export Templates. Click Download and Install for your current Godot version. Wait for it to finish.

💡 This only needs to be done once per Godot installation.
2

Add a Web Export Preset

Go to Project → Export. Click Add... and select Web from the list.

3

Configure the Export

In the Web preset settings, make sure "Export Type" is set to HTML5. You can leave most settings as default for now.

⚠️ Make sure your main scene is set under Project → Project Settings → Application → Run → Main Scene
4

Export the Files

Click Export Project. Create a new folder called my-game-web and save the export there. Godot will generate these files:

index.html
index.js
index.wasm
index.pck
index.png (icon)
✅ Your game is now exported! Keep this folder — you'll upload all these files to itch.io.
🛠️ Phase 1 — Export Your Unity Game for Web
1

Install WebGL Build Support

Open Unity Hub → Installs. Click the three dots on your Unity version → Add Modules. Check WebGL Build Support and install it.

2

Switch Platform to WebGL

In Unity go to File → Build Settings. Select WebGL from the list. Click Switch Platform and wait for Unity to reimport assets.

💡 This can take a few minutes the first time.
3

Build the Game

In Build Settings, click Build. Choose a folder like WebBuild. Unity creates a folder with HTML and WebGL files inside.

⚠️ In Player Settings, set Compression Format to Disabled — itch.io needs this to work properly.
4

Zip the Build Folder

Go to your WebBuild folder. Select all files inside it, right-click and Compress to ZIP. Name it my-unity-game.zip.

✅ Your ZIP file is ready to upload to itch.io!
🛠️ Phase 1 — Export Your Unreal Game for Web
1

Important Note About Unreal + Web

Unreal Engine 5 does not officially support HTML5/WebGL export. The best option for Unreal games on itch.io is to export for Windows as a downloadable game.

⚠️ Players will need to download and run your game — it won't play in the browser. This is normal for Unreal games on itch.io.
2

Package for Windows

Go to Platforms → Windows → Package Project. Choose an output folder. Unreal will build your game into a .exe with supporting files.

3

Zip the Package

Select the entire packaged folder, right-click → Compress to ZIP. Name it my-unreal-game.zip.

✅ Your ZIP is ready to upload to itch.io as a downloadable game!
🌐 Phase 2 — Upload to itch.io
1

Create a Free itch.io Account

Go to itch.io and click Register. It's completely free — no credit card needed.

2

Create a New Project

Click your profile icon → Dashboard → Create new project.

3

Fill In Your Game Details

Fill in the Title, Short description, and pick a Cover image for your game. Under Kind of project select HTML (for Godot/Unity web) or Downloadable (for Unreal).

4

Upload Your Game Files

Scroll to Uploads. Click Upload files and upload your ZIP file or individual HTML files.

💡 For Godot web exports, upload all the exported files (html, js, wasm, pck). For Unity/Unreal, upload the ZIP.
5

Enable the Embed (Godot & Unity only)

After uploading, check the box that says "This file will be played in the browser". Set the viewport size to match your game (e.g. 1280x720).

6

Set Visibility to Public

Under Visibility & access, select Public. Then scroll down and click Save & view page.

✅ Your game is now live on itch.io! Copy your game page URL — you'll need it for the next step.
https://yourusername.itch.io/your-game-name
🚀 Phase 3 — Submit to Raking Productions
1

Click "Submit Game" on This Site

Click the + Submit Game button in the top navigation bar above.

2

Fill In Your Game Info

Enter your game name, select your engine, paste your itch.io game URL, and add a short description. Pick an emoji icon for your game thumbnail.

3

Submit!

Click Submit Game. Your game will immediately appear in the Games section and players can click it to go directly to your itch.io page to play!

🎉 That's it! Your game is now listed on Raking Productions and playable via itch.io!
